== Common Parsing Patterns

=== Using the Unified Parser

LutaML provides a single entry point for all formats:

[source,ruby]
----
require "lutaml"

# Automatically detects format and parses
document = Lutaml::Parser.parse([File.new("model.xmi")])
document = Lutaml::Parser.parse([File.new("model.qea")])
document = Lutaml::Parser.parse([File.new("schema.exp")])
----

=== Error Handling

All parsers follow consistent error handling:

[source,ruby]
----
begin
  document = Lutaml::Parser.parse([File.new("model.xmi")])
rescue Lutaml::ParseError => e
  puts "Parse error: #{e.message}"
end
----
