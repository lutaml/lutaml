== Welcome to LutaML

LutaML is a universal Ruby library and CLI for accessing, parsing, and manipulating data models across multiple modeling languages.

=== What LutaML Does

LutaML provides a unified interface for:

* **UML Model Access**: Parse Enterprise Architect (EA) models from QEA (SQLite) and XMI formats
* **EXPRESS Support**: Handle ISO 10303-11 EXPRESS schemas used in industrial standards
* **Diagram Generation**: Generate pixel-perfect SVG diagrams from EA models with full styling
* **Model Transformation**: Convert between different modeling formats
* **Repository Management**: Store and query models in compressed LUR (LutaML UML Repository) format
* **Documentation Generation**: Create browsable SPA (Single Page Application) sites from models
